## Installation

### Dashboard

To install the client, use the following command:

```bash
npm i
```

To run the client, use the following command:

```bash
npm run dev
```

### Server

To install the server, use the following command:

```bash
npm i
```

To run the server, use the following command:

```bash
npm run dev
```
